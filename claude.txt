### Technical Knowledge for Claude Applications

#### Vehicle Troubleshooting Knowledge Base
```
SYSTEMATIC_DIAGNOSIS_APPROACH:
1. Safety Assessment
   - Ensure vehicle is on level ground and parking brake engaged
   - Check for obvious safety hazards before diagnosis
   - Verify power is off before electrical inspection

2. Basic System Checks
   - Battery charge level and connections
   - Key switch and control system function
   - Motor and drivetrain inspection
   - Brake and steering system verification

3. Advanced Diagnostics
   - Electrical system testing with multimeter
   - Motor controller diagnostics
   - Battery individual cell testing
   - Charging system analysis

4. Solution Implementation
   - Prioritize safety-related issues first
   - Use genuine parts when available
   - Document all service performed
   - Test thoroughly before customer return
```
